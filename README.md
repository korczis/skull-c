# skull-c

Learning materials for C language

See [skull](https://github.com/korczis/skull)

## Prerequisites

- Decent Unix/Linux 
- gcc/clang
- cmake

## Getting started

```
git clone https://github.com/korczis/skull-c.git
cd skull-c
make
```
